# Movie Data Analysis: Identifying Key Drivers of Profitable Films | 映画データ分析: 収益性の高い映画の要因分析

## Overview | 概要
This project analyses movie metadata to identify key factors that contribute to a film's profitability.
The goal is to generate actionable insights to support decision-making in movie production.
本プロジェクトでは、映画のメタデータを分析し、収益性に影響を与える要因を特定します。
映画製作における意思決定に活用できるインサイトの創出を目的としています。

## Dataset | データセット
Source: IMDb movie metadata (movie_metadata.csv)
Includes:
- Budget / Revenue
- Genre
- IMDb score
- Director / Actor information

## Obbjectives | 目的
- Identify factors influencing movie profitability
- Analyse relationships between budget, genre, and profit
- Evaluate impact of director/actor experience
- Provide business recommendations

- 映画の収益性に影響する要因の特定
- 予算・ジャンル・利益の関係分析
- 監督・俳優の経験が与える影響の評価
- ビジネスへのインサイトの提供

## Tech Stack | 使用技術
- Python (Pandas, NumPy, Matplotlib)

## Methodolofy | 分析手法
- Data cleaning and preprocessing
- Feature engineering
- Exploratory Data Analysis
- Statistical testing

- データ前処理・クリーニング
- 特徴量作成
- 探索的データ分析
- 統計的検定

## Key Insights | 主な分析結果
- Colour films tend to generate higher profits
- Genre selection has a strong impact on profitability
- Low-budget films in specific genres can be highly profitable
- Director experience increases likelihood of success

- カラー映画は白黒映画より高い収益を生みやすい
- ジャンル選択が収益性に大きく影響する
- 特定ジャンルでは低予算でも高収益が可能
- 経験豊富な監督ほど成功確率が高い

## Business Insights | 提案
- Focus on high-performing genres (e.g. Family, Adventure, Music)
- Avoid conflicting genre combinations
- Hire experienced directors
- Prioritise colour film production

- 高収益が期待できるジャンル（Family、Adventure、Musicなど）に注力
- 相性の悪いジャンルの組み合わせを避ける
- 経験豊富な監督の起用
- カラー映画の制作を優先
